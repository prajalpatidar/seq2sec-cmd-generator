================================================================================
                              PACKAGE OVERVIEW
================================================================================

Two deployment options available:

1. C++ DEPLOYMENT (deployment-cpp/)
   - No Python required ✓
   - Size: 44 MB (30 MB compressed)
   - Memory: 45-55 MB RAM
   - Speed: 60-120ms inference
   - Best for: RDK-B embedded devices

2. PYTHON DEPLOYMENT (deployment-python/)
   - Python 3.8+ required
   - Size: 76 MB (71 MB compressed)
   - Memory: 100-150 MB RAM
   - Speed: 80-150ms inference
   - Best for: Development, flexibility

================================================================================
                           C++ DEPLOYMENT GUIDE
================================================================================

QUICK START:
  cd deployment-cpp
  ./run.sh generate "show network interfaces"

TRANSFER TO DEVICE:
  tar -czf seq2seq-cpp.tar.gz deployment-cpp/
  scp seq2seq-cpp.tar.gz user@device:/tmp/
  ssh user@device
  cd /tmp && tar -xzf seq2seq-cpp.tar.gz
  cd deployment-cpp && ./run.sh generate "test"

USAGE EXAMPLES:
  ./run.sh generate "show network interfaces"    # → ifconfig
  ./run.sh generate "list all files"             # → ls -l
  ./run.sh generate "show wifi ssid"             # → dmcli ert getv ...
  ./run.sh interactive                           # Interactive mode
  ./run.sh batch input.txt output.txt            # Batch processing

CONTENTS:
  - cmd_generator (776 KB)           Executable
  - libseq2sec_lib.so (1.3 MB)       Shared library
  - models/onnx/ (25 MB)             ONNX models
  - models/checkpoints/ (5 KB)       Vocabularies
  - onnxruntime/ (6.3 MB)            Runtime library
  - run.sh                           Launcher script
  - README.md                        Documentation

REQUIREMENTS:
  - Linux x86_64 or ARM64
  - GLIBC 2.31+ (Ubuntu 20.04+)
  - No Python required ✓

================================================================================
                         PYTHON DEPLOYMENT GUIDE
================================================================================

QUICK START:
  cd deployment-python
  python3 -m venv venv
  source venv/bin/activate
  pip install -r requirements.txt
  pip install -e .
  python -m cli.cmd_generator generate "show network interfaces"

USAGE EXAMPLES:
  python -m cli.cmd_generator generate "show network interfaces"
  python -m cli.cmd_generator generate "show wifi ssid"
  python -m cli.cmd_generator interactive

CONTENTS:
  - cli/                             CLI tool
  - models/ (50 MB)                  PyTorch + ONNX models
  - scripts/                         Utilities
  - requirements.txt                 Dependencies
  - setup.py                         Installation script
  - README.md                        Documentation

REQUIREMENTS:
  - Python 3.8+
  - torch>=2.5.1
  - numpy>=1.24.3
  - onnx>=1.17.0
  - onnxruntime>=1.16.3
